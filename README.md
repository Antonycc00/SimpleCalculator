A minimal and easy-to-use Command Line Interface (CLI) calculator.

## Features
* **Basic Arithmetic:** Addition, subtraction, multiplication, and division.
* **Simple Interface:** Clean and straightforward command-line input.

## How to Install

1. Clone this repository:
```bash
   git clone [https://github.com/yourusername/calculator.git](https://github.com/yourusername/calculator.git)
Navigate into the project folder:

Bash
   cd calculator
How to Use
Run the calculator using Python:

Bash
python calculator.py
